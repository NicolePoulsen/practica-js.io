JAVASCRIPT

DEFINICIÓN

JavaScript es un lenguaje de programación interpretado, dialecto del
estándar ECMAScript. Se define como orientado a objetos,basado en
prototipos, imperativo, débilmente tipado y dinámico.

HISTORIA

JavaScript es un lenguaje de programación nacido hace más de 20 años.
Para ser exactos en el año de 1995 dentro del proyecto para el navegador
Netscape Navigator con el objetivo de hacer la web más dinamica e
interactiva.
Tal y como fue concebido inicialmente, los programas hechos en JavaScript
son ejecutados en el navegador web y, por tanto, el navegador del usuario
y no en el servidor donde se encuentra hospedado el sitio.
